# Emotion Detection System

## Project Description
This is a Python-based application that uses the **Watson NLP library** to analyze text and detect various emotions such as anger, disgust, fear, joy, and sadness.

## Tasks Performed:
1. Created an emotion detection function using Watson NLP.
2. Formatted the output to show individual emotion scores.
3. Identified the dominant emotion from the analyzed text.
4. Developed a Flask web application to provide a user interface.
